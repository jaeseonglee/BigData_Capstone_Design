## 프로젝트 개요
- 최근 반려동물을 키우는 사람들이 많아지고 있었고 코로나 사태로 인해 더욱 증가하였습니다. IT 기술이 발전함에 따라서 반려 동물을 케어할 수 있는 서비스가 제시되고 있으며 스마트 IoT 기술을 이용한 지능적인 기능이 최근 널리 사용되고 있다. 팀 ACM은 __스마트 IoT 기술을 활용하여 반려 동물들을 케어할 수 있는 시스템__ 을 제안합니다.
